# LocalHub v1.7 기능 변경

- 홈·지역정보 상세/목록 하단의 중복 데이터 출처 프레임 제거
- 게시글 작성 화면의 평문 비밀번호 안내 문구 사용자 화면에서 제거
- 실시간 알림별 삭제 버튼 추가
- 알림 목록은 한 번에 최대 5개 높이만 노출하고 초과분은 스크롤 제공
- 마우스/포인터 클릭 후 드래그 방식으로 알림 목록 스크롤 가능
- 지역정보 카드에서 북마크 등록·해제 가능
- 지역정보 검색 영역에 북마크만 보기 필터 추가
- 북마크 필터는 서버의 X-Client-Key 기준으로 정확한 전체 목록과 페이지네이션 제공
