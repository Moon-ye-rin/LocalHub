# 데이터 출처 및 라이선스

## 출처

- 제공기관: 한국관광공사
- 데이터명: 국문 관광정보 서비스(TourAPI 4.0)
- 공공데이터포털: https://www.data.go.kr/data/15101578/openapi.do
- 적용 지역: 서울, 경기
- 실제 적재 건수: 12,512건
- 실제 제공 파일: 서울 8개, 경기 8개

## 라이선스

- 공공누리 제3유형
- 출처 표시 필수
- 원본 데이터 변경 금지
- 상업적 이용 허용
- 라이선스 조건 준수 시 재배포 허용

## 구현 원칙

- 원본 필드명과 값을 유지합니다.
- 빈 문자열도 NULL로 임의 변경하지 않습니다.
- API 명세가 허용한 `mapx`, `mapy`만 SQLite REAL로 변환합니다.
- `firstimage`가 빈 문자열 또는 NULL인 항목은 사용자 화면과 챗봇에서 제외합니다.
- 목록, 상세, 챗봇 답변에 출처 문구를 표시합니다.

## 화면 표기 문구

> 이 서비스는 한국관광공사 TourAPI(TourAPI 4.0)의 데이터를 활용하였습니다. 출처: 한국관광공사. 라이선스: 공공누리 제3유형.

## 지도 및 A* 길찾기 도로 데이터

- 제공 프로젝트: OpenStreetMap
- 지도 타일: `https://tile.openstreetmap.org`
- 도로망 조회: Overpass API (`https://overpass-api.de/api/interpreter`)
- 라이선스: Open Database License(ODbL) 1.0
- 화면 표기: `지도·도로 데이터 © OpenStreetMap contributors`

### 구현 원칙

- 출발·도착 장소의 `mapx`, `mapy`를 기준으로 필요한 구간의 도로 노드와 way만 실행 시점에 조회합니다.
- 조회한 도로망은 A* 경로 계산에만 사용하고 SQLite에는 저장하지 않습니다.
- 동일 구간의 반복 요청을 줄이기 위해 서버 메모리에 최대 20분 동안만 임시 캐시합니다.
- 계산 결과는 예상 경로이며 실시간 교통, 공사, 돌발 상황은 반영하지 않습니다.
