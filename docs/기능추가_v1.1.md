# LocalHub 기능 추가 v1.1

## 추가·수정 항목

1. **댓글 기능**
   - `comments` SQLite 테이블 추가
   - 게시글별 댓글 목록·작성·수정·삭제 API 추가
   - 댓글 작성 시 수정·삭제용 비밀번호 등록
   - 게시글 상세 화면에서 댓글 작성, 인라인 수정, 비밀번호 확인 삭제 지원

2. **페이지네이션 개선**
   - 전체 페이지 버튼을 모두 렌더링하지 않고 현재 묶음의 최대 10개 번호만 표시
   - 좌·우 화살표로 이전·다음 10페이지 묶음 이동

3. **내부 지도 시각화**
   - 외부 지도 링크 버튼 제거
   - TourAPI의 `mapx`, `mapy` 좌표를 활용해 상세 페이지 내부에 OpenStreetMap 삽입
   - 좌표가 없을 경우 안내 상태 표시

4. **게시글 이미지 첨부**
   - 작성·수정 폼에서 최대 5개 파일 선택 및 미리보기
   - JPG/JPEG/PNG/GIF, 파일당 5MB 제한
   - 게시글 저장 후 `/api/posts/{post_id}/images`로 업로드
   - 백엔드 상대 경로 이미지를 API 서버 절대 경로로 변환해 표시

5. **게시글 작성 폼 스타일 복구**
   - 제목·내용·비밀번호 입력 요소에 공통 `form-input` 스타일 적용
   - 폼 레이아웃, 라벨, 포커스 상태, 모바일 반응형 정리

## 댓글 API

```text
GET    /api/posts/{post_id}/comments
POST   /api/posts/{post_id}/comments
PUT    /api/posts/{post_id}/comments/{comment_id}
DELETE /api/posts/{post_id}/comments/{comment_id}
```

댓글 작성·수정 요청:

```json
{
  "content": "댓글 내용",
  "password": "1234"
}
```

댓글 삭제 요청:

```json
{
  "password": "1234"
}
```
