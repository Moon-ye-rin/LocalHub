# LocalHub v1.10.2 지도·경로 UI 수정

- 지역정보 상세의 지도 영역을 `위치`와 `경로찾기` 탭으로 분리했습니다.
- 최초 진입 시 v1.9.1 방식의 OpenStreetMap 위치 지도를 기본으로 표시합니다.
- 경로찾기 탭에서는 출발지·도착지 핀과 계산된 경로를 표시합니다.
- 지도 타일이 차단되어도 핀과 경로가 보이도록 중립 배경을 유지합니다.
- `A* 계산 중...` 문구를 `탐색 중...`으로 변경했습니다.
- 지도 범례의 `A* 경로`를 `경로`로 변경했습니다.
- 이동 방식 선택 UI와 탐색 방식 결과를 제거했습니다.
- 결과 요약을 경로 거리, 예상 시간(자동차), 예상 시간(도보)로 구성했습니다.
