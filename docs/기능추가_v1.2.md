# LocalHub 기능 보완 v1.2

## 지역정보 카드 정렬
- 데스크톱 3열 그리드에 맞춰 페이지당 조회 수를 20개에서 21개로 변경했습니다.
- 일반 페이지는 7행 × 3열로 빈칸 없이 표시됩니다.
- 전체 데이터가 끝나는 마지막 페이지나 검색·필터 결과가 부족한 경우에는 남은 데이터만 표시됩니다.
- 페이지 번호는 기존처럼 10개 단위로 묶어 좌우 화살표로 이동합니다.

## 게시글 수정 시 기존 이미지 삭제
- 수정 화면의 기존 첨부 이미지마다 삭제(X) 버튼을 추가했습니다.
- 게시글 수정용 비밀번호를 서버에서 다시 검증한 뒤 DB 레코드와 실제 업로드 파일을 함께 삭제합니다.
- 신규 이미지는 기존 이미지를 포함해 최대 5개까지 첨부할 수 있습니다.

## 추가 API
- `DELETE /api/posts/{post_id}/images`
- 요청 본문: `{ "image_url": "...", "password": "..." }`
