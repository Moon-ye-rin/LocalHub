# LocalHub v1.10.1 지도 로딩 오류 수정

## 원인
v1.10 지도 컴포넌트가 실행 시 `unpkg.com`에서 Leaflet JavaScript/CSS를 동적으로 내려받도록 되어 있었다.
학교·사내 네트워크 또는 보안 프록시가 해당 CDN 접속을 차단하면 `Leaflet을 불러오지 못했습니다.` 오류가 발생하고, 기존 위치 핀 지도까지 함께 표시되지 않았다.

## 수정
- Leaflet 외부 JavaScript/CSS 의존성을 제거했다.
- Vue 컴포넌트 내부에서 OpenStreetMap 표준 타일을 직접 배치하는 경량 지도 렌더러를 구현했다.
- 출발지·도착지 핀과 A* 경로 polyline을 동일 지도 위에 표시한다.
- 마우스 드래그 이동, 확대·축소, 출발/도착 맞춤 기능을 제공한다.
- 지도 타일 로딩이 차단되면 기존 OpenStreetMap iframe 위치 지도로 자동 대체한다.

## 참고
A* 경로 계산은 기존 FastAPI 백엔드 구현을 그대로 사용한다. 지도 타일과 도로망 데이터 조회에는 인터넷 연결이 필요하다.
