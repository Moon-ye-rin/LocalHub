# LocalHub v1.3 데이터·지역 필터 보완

## 추가 데이터

- 서울 음식점 데이터 1,097건
- 경기 8개 콘텐츠 유형 데이터 4,897건
- 전체 원본 12,512건
- 대표 이미지가 있는 노출 대상 11,598건
  - 서울 7,102건
  - 경기 4,496건

## 구현 변경

1. `GET /api/locations`에 `region=서울|경기` 필터를 추가했습니다.
2. 지역정보 목록 화면과 홈 추천 영역에서 서울/경기 선택이 가능합니다.
3. `firstimage`가 빈 문자열 또는 NULL인 항목은 목록, 상세, 홈 추천, 챗봇에서 제외합니다.
4. 지역정보 페이지당 21개와 10페이지 단위 페이지네이션을 유지합니다.
5. 챗봇이 질문의 서울/경기 표현을 감지하여 해당 지역을 우선 검색합니다.
6. 데이터 적재기는 `app/data` 아래의 서울·경기 폴더를 재귀 탐색합니다.

## 보안

업로드 자료에 포함된 데이터 다운로드 스크립트에는 서비스 키가 하드코딩되어 있어 최종본에는 포함하지 않았습니다.
추가 수집이 필요할 경우 키를 `.env`의 별도 환경변수로 관리하는 방식으로 변경해야 합니다.
