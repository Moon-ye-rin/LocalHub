# LocalHub v1.10 — 지역정보 A* 길찾기

## 사용자 흐름

1. 지역정보 상세 페이지를 연다.
2. 현재 보고 있는 장소가 출발지로 자동 설정된다.
3. 도착지를 장소명·주소로 검색하거나 `가까운 장소 빠른 선택`에서 고른다.
4. 도보 또는 자동차를 선택한다.
5. `경로 찾기`를 누르면 같은 지도에 출발·도착 핀과 경로가 표시된다.

## 구현 구조

- Frontend: Leaflet 1.9.4를 공식 CDN에서 동적 로드
- 지도 타일: OpenStreetMap 표준 타일
- 도로 데이터: Overpass API의 `highway` way/node
- Backend: OSM 도로 노드를 인접 리스트 그래프로 구성
- Algorithm: Haversine 거리를 휴리스틱으로 사용하는 A*
- API: `POST /api/routes/astar`

## API 요청 예시

```json
{
  "start_contentid": "126508",
  "end_contentid": "999001",
  "mode": "walk"
}
```

`mode`는 `walk` 또는 `drive`이다. 응답에는 경로 좌표 배열, 총 거리, 예상 시간, 탐색 노드 수가 포함된다.

## 제약 및 예외 처리

- 기본 직선거리 45km 이내만 지원한다.
- 장소와 도로 노드의 거리가 너무 멀면 경로를 만들지 않는다.
- 도보에서는 자동차전용도로를 제외한다.
- 자동차에서는 보행로·계단·자전거도로 등을 제외하고 일방통행을 반영한다.
- 실시간 교통, 공사, 임시 통행 제한은 반영하지 않는다.
- 공용 Overpass API 상태에 따라 경로 계산 시간이 길어질 수 있다.

## 환경변수

```env
OVERPASS_URL=https://overpass-api.de/api/interpreter
ROUTE_TIMEOUT_SECONDS=45
ROUTE_MAX_DISTANCE_KM=45
ROUTE_MAX_SNAP_METERS=1500
ROUTE_MAX_GRAPH_NODES=120000
```
