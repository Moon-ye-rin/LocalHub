# LocalHub v1.5 기능 추가

## 1. 게시글 좋아요 토글

- 이미지 첨부 게시글에서도 좋아요 버튼이 정상 클릭되도록 이미지 영역의 포인터 이벤트와 버튼 레이어를 조정했습니다.
- `POST /api/posts/{post_id}/like`: 좋아요 등록
- `DELETE /api/posts/{post_id}/like`: 좋아요 취소
- 상세 조회 응답에 현재 브라우저의 `liked` 상태를 포함합니다.
- 중복 등록 방지는 `X-Client-Key`와 `UNIQUE(post_id, client_key)`로 유지합니다.

## 2. 지역정보 상세 개편

- 지도 아래의 콘텐츠 ID, 법정동 코드, 분류체계, 우편번호, 원본 수정 시각 표시를 제거했습니다.
- 현재 장소의 `mapx`, `mapy` 좌표와 가장 가까운 지역정보 5개를 계산해 카드로 표시합니다.
- 카드 선택 시 해당 지역정보 상세 페이지로 이동합니다.
- 대표 이미지가 없는 항목은 기존 정책대로 추천 대상에서도 제외합니다.

## 3. 지역정보 참여 기능

추가 테이블:

- `location_comments`: 닉네임, 댓글, 0~5점 별점
- `location_likes`: 브라우저별 좋아요
- `location_bookmarks`: 브라우저별 북마크
- `location_views`: 브라우저별 상세 조회

닉네임을 입력하지 않으면 `익명`으로 저장합니다. 별점은 별 버튼으로 0~5점을 선택할 수 있습니다.

주요 API:

- `GET/POST /api/locations/{contentid}/comments`
- `POST/DELETE /api/locations/{contentid}/like`
- `POST/DELETE /api/locations/{contentid}/bookmark`
- `GET /api/locations/{contentid}`: 조회 기록, 참여 통계, 근처 5곳 포함

## 4. 대시보드

- 인기 지역 순위 제목 아래의 점수 산정식 문구를 화면에서 제거했습니다.
- 인기 지역정보 순위를 추가했습니다.
  - 점수: 댓글 수 × 3 + 좋아요 수 × 2 + 조회 수
- 평점 높은 지역정보 순위를 추가했습니다.
  - 지역정보 댓글 별점 평균 내림차순
  - 동점일 때 별점 참여 수와 댓글 수를 우선 적용
