# LocalHub v1.4 게시글 지역·대시보드 추가

## 게시글 지역 입력

- 게시글 작성·수정 시 기존 입력 항목을 유지하면서 지역 드롭다운 2개를 추가했습니다.
- 1차 선택: 서울 / 경기
- 2차 선택: 서울 25개 자치구 또는 경기 31개 시·군
- 신규 게시글 API는 `region`, `district`를 필수로 검증합니다.
- 기존 SQLite DB에는 서버 시작 시 컬럼을 자동 추가하는 호환 마이그레이션을 적용했습니다.
- 게시글 카드와 상세 화면에 선택한 지역을 배지로 표시합니다.

## 대시보드

- 경로: `/dashboard`
- API: `GET /api/dashboard`
- 상단 총 게시물·조회수·좋아요·등록 장소 KPI 카드는 요구에 따라 제외했습니다.
- 최근 7개월의 관광지·맛집·문화·축제 게시글을 누적 막대로 표시합니다.
- 전체, 서울, 경기의 카테고리 분포를 각각 도넛 그래프로 표시합니다.
- 인기 지역 순위는 `(댓글 수 × 3) + (좋아요 수 × 2) + 조회 수` 점수를 사용합니다.
- 주간 트렌드는 `post_views.viewed_at`, `post_likes.created_at`의 실제 이벤트를 현재 주 월~일 기준으로 집계합니다.

## DB 변경

- `posts.region VARCHAR(10)`
- `posts.district VARCHAR(30)`
- `idx_posts_region_district(region, district)`
- 지역 변경 시에도 `updated_at`이 갱신되도록 트리거 보완
