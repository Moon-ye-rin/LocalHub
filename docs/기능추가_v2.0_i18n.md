# LocalHub v2.0 — 한/영 전환 및 다국어 지원

## 적용 기능

- Vue I18n Composition API 기반 한국어/영어 전환
- 상단 내비게이션의 `KO / EN` 언어 선택 버튼
- 선택 언어를 `localStorage(localhub-locale)`에 저장하여 새로고침 후 유지
- 내비게이션, 홈, 게시판, 지역정보, 게시글 작성/상세, 댓글, 대시보드, 챗봇, 공유, 오류/빈 상태 문구 번역
- 날짜와 알림 시각을 선택 언어의 로케일 형식으로 표시
- 카테고리, 서울/경기, 서울 자치구 및 경기 시·군 영문 표기
- 영어 챗봇 질문 지원 및 영어 답변 제공
- OG locale을 `ko_KR / en_US`로 동적 적용

## TourAPI 원본 콘텐츠 처리

TourAPI 데이터는 공공누리 제3유형(출처표시 + 변경금지)이 적용되어 있습니다.
변경금지 조건에는 번역을 통한 2차적 저작물 작성도 포함되므로 장소명과 주소 원문은 임의 번역하지 않고 한국어 원문을 유지합니다.
영어 모드에서는 화면 UI, 카테고리, 지역 구분, 설명 및 챗봇 안내를 영어로 제공하면서 원본 장소명·주소를 함께 표시합니다.
